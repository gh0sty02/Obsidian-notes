## Creative superpowers for Claude

Connect Pixa to Claude and unlock professional image and video tools — right inside Claude, Cowork, and Claude Code.

Claude Instructions

1

Open [Claude](https://claude.ai/)

2

Go to Settings → Connectors → Add Custom Connector

3

Paste this URL:

`https://mcp.pixa.com/mcp`

[View Documentation→](https://www.pixa.com/docs/mcp)

Plug & Play

### One config, every creative tool

Add the Pixa MCP server once and Claude automatically discovers all available tools. No API keys to juggle, no SDK to learn — just connect and create.

Available tools

Remove Background— Remove image backgrounds

Generate Image— Generate images from text

Upscale— Upscale images 2–4x

Generate Video— Generate video from text or image

Erase Object— Erase objects from images

Asset Library— Browse your asset library
