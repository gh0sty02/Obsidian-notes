## Patterns Documented (with Code Examples)

1. **Sliding Window** — dynamic window on strings/arrays; signals: "longest", "shortest", "contains" on contiguous subarray
2. **Two Pointers** — sorted arrays, pair/triplet sums, opposite-end traversal
3. **Fast & Slow Pointers** — linked list cycles, finding middle, Nth from end
4. **Merge Intervals** — always sort by start time first; overlapping intervals
5. **Cyclic Sort** — numbers in [0,n] or [1,n] range; find missing/duplicate

## Cheat Sheet (Pattern Recognition)

| Trigger | Pattern |
|---------|---------|
| Max subarray, increase or restart | Kadane |
| Range queries / subarray counts | Prefix Sum |
| Partition, flag-style sorting | Dutch National Flag |
| Pair in sorted array | Two Pointers |
| Grow/shrink subarray | Sliding Window |
| Increasing/decreasing sequence | Monotonic Stack |
| Sorted array / binary answer | Binary Search |
| Top K / greedy ordering | Heap |
| All possible solutions | Backtracking |
| Prefix search in strings | Trie |
