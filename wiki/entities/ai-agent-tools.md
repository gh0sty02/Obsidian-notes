Curated tools for building, extending, and automating with AI agents. Sourced from `raw/Clippings/`.

## Memory Systems

### MemPalace
- **Repo:** `github.com/MemPalace/mempalace`
- Claims highest benchmark score for AI memory systems
- Free and open-source
- Designed to give LLMs persistent, structured memory across sessions

---

## Research & Synthesis Agents

### Last30Days Skill (`mvanhorn/last30days-skill`)
- **Repo:** `github.com/mvanhorn/last30days-skill`
- AI agent skill that researches any topic across Reddit, X, YouTube, HN, Polymarket, and the web
- Synthesizes a grounded summary of what happened in the last 30 days
- Useful for staying current on fast-moving technical topics

## Orchestration Frameworks

### Paperclip (`paperclipai/paperclip`)
- **Repo:** `github.com/paperclipai/paperclip`
- Open-source orchestration for "zero-human companies"
- Goal: automate full business workflows with AI agents

## Job Search Automation

### AIHawk (`feder-cr/Jobs_Applier_AI_Agent_AIHawk`)
- **Repo:** `github.com/feder-cr/jobs_applier_ai_agent_aihawk`
- Automates job applications using AI
- Tailors applications per job listing
- Core architecture open source; third-party provider plugins removed for copyright
